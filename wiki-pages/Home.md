# Emacs AI Study Group - Wiki

Welcome to the Emacs AI Study Group wiki! This resource provides comprehensive documentation about the various AI tools and integrations available for Emacs.

## Overview

This repository demonstrates and documents five major AI integration tools for Emacs, each with different strengths and use cases:

- **[gptel](Gptel)** - Simple, multi-provider LLM client
- **[org-ai](Org-AI)** - AI capabilities within org-mode
- **[Ellama](Ellama)** - Ollama-specific integration
- **[aider.el](Aider-el)** - AI pair programming interface
- **[elisp-dev-mcp](Elisp-Dev-MCP)** - Model Context Protocol server
